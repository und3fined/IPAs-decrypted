# IPAs decrypted

### How to find your IPA?

1. Copy link to App Store. Eg: `https://apps.apple.com/ru/app/lineage2m/id1575437656`
2. End of link show App Store Id. Eg: `id1575437656`
3. Copy it and find in this repo
